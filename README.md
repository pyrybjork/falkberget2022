### Help with translation

All the text in this project can be found as json object in src/data/content.ts.

Its formatted followingly:

```
{
  "internal_id": {
    "en" : "In English"
    "fi" : "Suomeksi"
    "sv" : "På svenska"
  }
}
```

Almost all of the Swedish translations are missing.
